# 종이의 개수
## 풀이방법
- 재귀함수로 풀이
1. 수의 자릿수를 모두 더하는 함수를 재귀함수로 구현
2. 문제에서 주어진 수를 계속 자리수를 더해가며 10이하의 수로 만드는 재귀함수 구현
3. 1번과 2번 함수를 사용하여 문제에서 주어진 수가 10 이하가 됐을 때 문제에서 요구하는 내용 확인 후 출력

- 입력값이 최대 1,000,000,000 **자리수**의 값이 올수 있음
    - 대충보고 1,000,000,000 이하의 수인줄 알고 풀어서 런타임 에러 발생 ...
- 재귀함수를 계속 사용하는것 보다는 루프문을 사용하는게 조금더 좋을것 같기는 한데, var를 사용하지 않고 풀어보기 위해 재귀함수 사용
